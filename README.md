# Fabric Documentation

Official documentation for [Fabric](https://github.com/farpointhq/Fabric) - AI-powered code assistant.

## Live Site

Visit the documentation at: https://docs.fabric.dev

## Development

### Prerequisites

- Python 3.x
- pip

### Setup

```bash
# Clone the repository
git clone https://github.com/farpointhq/Fabric-docs.git
cd Fabric-docs

# Install dependencies
pip install mkdocs-material mkdocs-glightbox

# Start local server
mkdocs serve
```

Open http://localhost:8000 to view the docs.

### Building

```bash
mkdocs build
```

Output is in the `site/` directory.

## Deployment

Documentation is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

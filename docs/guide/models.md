## Choosing a Model

### For Code Review

**Recommended: Claude 3.5 Sonnet**

- Excellent at understanding context
- Catches subtle bugs
- Suggests improvements

### For Quick Questions

**Recommended: Claude 3.5 Haiku or GPT-4o**

- Fast responses
- Good for simple explanations
- Cost-effective

### For Long Documents

**Recommended: Gemini 1.5 Pro**

- 1 million token context
- Great for analyzing codebases
- Handles multiple files well

### For Complex Reasoning

**Recommended: o1-preview**

- Deep thinking on hard problems
- Mathematical proofs
- Algorithm design

## Switching Models

1. Click the model dropdown
2. Select your desired model
3. Continue the conversation

!!! tip "Mid-Conversation Switching"
    You can switch models mid-conversation! The new model will see the full context.

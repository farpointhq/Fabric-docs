## Upgrading

Fabric auto-updates by default. To manually update:

1. Download the latest release
2. Replace the existing application
3. Launch Fabric

Your settings and chat history are preserved.

## Beta Program

Join the beta to get early access to new features:

1. Open Settings
2. Navigate to Updates
3. Enable "Beta Channel"

!!! warning "Beta Stability"
    Beta releases may contain bugs. Not recommended for critical work.
